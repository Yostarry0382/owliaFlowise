# 利用者向け機能一覧

## 概要

エージェント利用者（一般社員）向けの機能一覧です。
MVPでは利用者が便利にエージェントを使えることを最優先とします。

---

## カタログ機能

| # | 機能 | 概要 | 優先度 | MVP |
|---|------|------|--------|-----|
| B-1 | エージェント一覧表示 | 公開済みエージェントをカード/リスト形式で表示 | 高 | ✅ |
| B-2 | 検索 | キーワードでエージェント名・説明を検索 | 高 | ✅ |
| B-3 | タグフィルタ | カテゴリタグで絞り込み | 高 | ✅ |
| B-4 | 部門フィルタ | 作成部門・対象部門で絞り込み | 中 | - |
| B-5 | 用途フィルタ | FAQ/資料作成/調査 等の用途で絞り込み | 中 | - |
| B-6 | ソート | 人気順/新着順/名前順 | 中 | - |
| B-7 | エージェント詳細表示 | 説明・能力値・使用例・作成者情報 | 高 | ✅ |

## MVPスコープまとめ

### 含める機能（8機能）

| # | 機能 |
|---|------|
| B-1 | エージェント一覧表示 |
| B-2 | 検索 |
| B-3 | タグフィルタ |
| B-7 | エージェント詳細表示 |
| B-11 | チャット画面 |
| B-12 | 会話履歴表示 |
| B-17 | エージェント情報API |
| B-18 | エージェント実行API |

### 含めない機能（10機能）

| # | 機能 | 理由 |
|---|------|------|
| B-4 | 部門フィルタ | 初期はエージェント数が少ないため不要 |
| B-5 | 用途フィルタ | 同上 |
| B-6 | ソート | タグフィルタで代替可能 |
| B-8 | お気に入り登録 | 認証なしのため実装困難 |
| B-9 | お気に入り一覧 | 同上 |
| B-10 | 利用履歴 | 同上 |
| B-13 | 会話リセット | 画面リロードで代替可能 |
| B-14 | 会話履歴保存 | 認証なしのため実装困難 |
| B-15 | 評価ボタン | Phase 2以降 |
| B-16 | コメント送信 | Phase 2以降 |

---

## 備考

- **チャット実行のメイン環境**: Owlia-Portal / Owlia-Sprite
- **Fabrica内チャット**: 作成後すぐに試すための簡易版
- **認証なし**: お気に入り・履歴等のユーザー紐づけ機能はMVP対象外
